# Themes

Theme definitions live in `internal/tui/theme`.

Built-in themes:

- `arch`
- `mono`
- `dark`
- `light`
- `high-contrast`

Themes control:

- accent color
- muted text
- warnings
- errors
- success text
- separators
- selected rows
- source badges
- status text

`NO_COLOR`, `TERM=dumb` and missing terminal color support force plain text rendering.
