# Keyboard and Mouse

aurview is keyboard-first.

Keyboard movement:

- `j` / `k`: down/up
- `h` / `l`: focus movement
- `gg` / `G`: top/bottom
- `/`: search
- `n` / `N`: history
- `Ctrl+d` / `Ctrl+u`: half-page movement
- `Ctrl+f` / `Ctrl+b`: page movement
- `Enter`: copy package name
- `Esc`: blur/close
- `q`: quit
- `?`: help

Mouse support is optional:

- click search to focus
- click package row to select
- double-click package row to copy
- wheel over results to move selection
- wheel over details to scroll detail content
